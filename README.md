# Triple Espresso

## Proyecto Trple Espresso Coffe Shop

Este proyecto es una landing page para una cafetería que incluye 4 secciones, una cabecera que contiene una barra de navegación que te lleva a las demás secciones por medio de enlaces, una sección de resetas donde se utilizaron etiquetas iframe para insertar videos de youtube en la página, otra sección incluye un formulario en el cual se puede reservar una mesa y finalmente un pié de página con los enlaces de contacto para las redes sociales y el copyright de quien elaboró la página.

En un futuro me gustaría modificar esta página para hacer que el formulario sí envíe de manera efectiva la información para generar una reserva, así mismo me gustaría utilizar medidas responsivas para que los bloques y elementos de la página reaccionen a los diferentes tamaños de pantalla de los distintos dispositivos desde los cuales se puede visitar un sitio web.
